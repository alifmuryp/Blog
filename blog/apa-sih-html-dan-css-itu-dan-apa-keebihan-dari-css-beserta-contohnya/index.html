Apa sih HTML dan CSS itu? Dan apa keebihan dari CSS? Beserta Contohnya

Hello world!

Kali saya akan membahas sedikit mengenai CSS dan HTML. Bagi kalian yang belum mengetahui apa itu HTML dan CSS atau sudah tahu tapi bingung nyari codenya dimana.


Pengertian

CSS (Cascading Style Sheets) jika diartikan kedalam bahasa indonesia berarti lembar penataan menurun. Dimana secara konteks CSS digunakan untuk memodifikasi Mark-up language(HTML).

Mark-up language (bahasa markah) merupakan bahasa markup dan kebanyakan digunakan untuk membuat website. Contohnya yaitu HTML yang mungkin dari kalian sudah familiar. Dimana bahasa ini lazim digunakan disetiap web browser.


Hubungan CSS dan HTML

Hubungannya kayak kamu sama doi, kalau punya. Heheh... Bercanda. Hubungannya yaitu kita jadi tak perlu menulis berulang kode style HTML yang sama, seperti contohnya jika kita ingin membuat semua bold menjadi warna merah, kita tak perlu menambahkan satu persatu style ke dalam HTML kita hanya perlu melakukan sekali saja didalam CSS.

Contoh :

HTML only

<b>Ini bold</b> 

<b>Ini bold</b>

<b>Ini bold</b>


Menjadi


 <b style="color:red;">Ini bold</b>

<b style="color:red;">Ini bold</b>

<b style="color:red;">Ini bold</b>


Jika menggunakan CSS

<style> b{color:red;}</style>

<b>Ini bold</b> 

<b>Ini bold</b>

<b>Ini bold</b>



Sedikit Ringkasan Mengenai CSS

Jika kalian tertarik dan ingin belajar CSS kalian bisa lihat di Youtube atau web. Dimana rekomendasi untuk belajarnya beserta roadmapnya ada disini.


NB : umumnya CSS ditaruh didalam template atau tema sebuah web. Dimana letaknya diatas </head>


Cara Kerja

Bisa dilihat pada contoh diatas dimana, CSS Mengubah semua tag HTML. Dimana dulu ketika tak ada CSS itu menjadi mimpi buruk. Bagaimana tidak jika kita ingin mengganti warna atau style lainnya, kita perlu mengganti semuanya. Dimana jika menggunakan css itu takperlu. Selain itu ada hal yang tidak bisa dilakukan oleh HTML namun CSS bisa.


Peran

Peran dari apa yang kupahami dan kesimpulan diatas. Kita tahu bahwa tanpa adanya CSS selain menyulitkan juga membut website menjadi tidak menarik. Selain itu juga tanpa afanya CSS akan membuat loading lama.


Keuntungan

  • Memudahkan front end untuk mengubah ataupun membuat tampilan yang menarik
  • Memperindah Web/blog agar lebih menarik.
  • Memperingan loading blog dan ukuran sebuah website.
  • Mudah untuk memperbaiki aebuah keasalahan atau bug.(bagi kalian yang sering kelupaan titik koma hehehe)
  • Pengembangan web lebih luas atau bervariasi.


Apakah kalian tahu Framework jika belum tahu klick disini


Macam-macam


Inline style sheet

Untuk Inline style sheet sendiri biasanya dibutuhkan jika halaman tertentu saja yang diberi CSS. Contohnya kita bisa menaruhnya di file langsung dengan menambahkan tag <style>. Seperti contoh diatas.


External style sheet

Aedangkan External style sheet biasanya diletakkan di file tersendiri. Jika anda pernah melihat <link rel... Itulah External style sheet. Meskipun tidak hanya CSS saja namun juga JavaScript juga.


Embedded style sheet

Untuk yang terakhir ini biasanya ada didalam halaman template. Bagi kalian yang menggunakan Blogger atau Wordpress, bisa dilihat di template atau tema, disana ada style yang terhubung ke semua halaman web. Meskipun kalian juga bisa tampil di halaman tertentu saja. Dan biasanya CSS ini ditaruh diatas kode </head> sehingga bisa disebut internal CSS. 


Pendapatku dan Kesimpulan

CSS itu sangat kita pelukan, karena kemudahan. Mulai dari bahasanya yang mudah dipahami karena hanya menggunakan bahasa inggris dan tak banyak peraturan yang membingungkan seperti bahasa pemrograman seperti js, python dll.
